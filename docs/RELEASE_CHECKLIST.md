### 2. Apple Developer account — **S$128/year**
- [ ] Go to **https://developer.apple.com/programs/enroll/**.
- [ ] Enroll as an **Individual** (simpler) or **Organization** (needs D-U-N-S number, ~2 weeks review).

- [ ] Note your **Apple Team ID** (10-char, at developer.apple.com → Membership). Put it in `eas.json` → `submit.production.ios.appleTeamId`.

### 3. Google Play Console account — **S$35 one-time**
- [ ] Sign up at **https://play.google.com/console/signup**.
- [ ] Complete identity verification (government ID + selfie — takes 2–3 days).
- [ ] Create a new app:
  - **App name:** Puppy Prep
  - **Default language:** English (Singapore) or en-GB
  - **App or game:** App
  - **Free or paid:** Free (with in-app purchases)
- [ ] Generate a **Service Account key** for automated submission:
  1. Go to Google Cloud Console → "APIs & Services" → "Credentials"
  2. Create a service account named `puppyprep-eas`
  3. Grant role: "Service Account User"
  4. Create key → JSON → download
  5. In Play Console → "Users and permissions" → "Invite new user" → paste the service account email → grant "Release manager" permission
  6. Save the JSON file as `credentials/google-play-service-account.json` in this repo (it's in `.gitignore`)

### 4. RevenueCat account (optional but recommended) — **free up to $10K/mo**
- [ ] Sign up at **https://www.revenuecat.com**.
- [ ] Create a project "Puppy Prep".
- [ ] Create a product: `puppyprep_pro_lifetime` — S$12.99 one-time.

## 🟦 Phase D — First build + internal test (once Phases A–C complete)

### Run once to configure:
```bash
npx eas login                           # use your Expo credentials
npx eas init                            # creates extra.eas.projectId in app.json
npx eas credentials                     # configures iOS + Android signing
```

### Build for internal test:
```bash
npx eas build --profile preview --platform ios        # internal TestFlight
npx eas build --profile preview --platform android    # internal APK you can sideload
```

Builds take 15–25 minutes each. EAS runs them in their cloud.

### Submit first TestFlight build:
```bash
npx eas submit --profile production --platform ios --latest
```

Apple does a quick automated review for TestFlight (usually 24 hours). Then invite yourself + friends via email.

### Submit first internal-track Play build:
```bash
npx eas submit --profile production --platform android --latest
```

Play Console internal track is live in ~15 minutes for approved testers.
